What is CSS?

CSS stands for Cascading Style Sheets. It's a stylesheet language that allows web developers to describe the presentation of a document written in HTML or XML.

CSS enables you to change colors, fonts, layouts, spacing, and even add animations to make a webpage feel alive and professional.

In CSS, styles are applied to HTML elements in three main ways:

  1. External CSS (linked in a separate file)
  2. Internal CSS (defined within the head section)
  3. Inline CSS (added directly to the HTML elements)

You can learn more at MDN Web Docs.
